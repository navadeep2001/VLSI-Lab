Hardware Data Types

These are used for designing hardware and represent actual circuit elements.

Data Type Size Range Usage
reg User-defined 4-state (0,1,X,Z) Procedural assignments (always, initial)
wire User-defined 4-state (0,1,X,Z) Continuous assignments (assign)
logic User-defined 4-state (0,1,X,Z) Can replace reg and wire

Example Usage


logic [7:0] data;  // 8-bit logic signal
reg [3:0] count;   // 4-bit register
wire clk;          // Wire used for continuous assignments
assign clk = 1'b1; // Assign a constant value

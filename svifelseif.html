In SystemVerilog, if-else if statements are used for conditional branching. These allow you to check multiple conditions in a sequential manner and execute code based on the first condition that evaluates to true.

    // Code block executed if condition_2 is true
end else if (condition_3) begin
    // Code block executed if condition_3 is true
end else begin
    // Code block executed if none of the above conditions are true
end

  • Finally, if neither, the student fails.

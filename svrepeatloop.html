The repeat loop is used to execute a block of code a fixed number of times. It's ideal when you know beforehand how many times something needs to be repeated and when you don‚Äôt need to use a loop index variable.

üìå Syntax

Single Statement:

Multiple Statements:

  statement 1;

üß† When to Use:

  • When the number of repetitions is known.
  • When you don‚Äôt need a loop counter variable (like i in for loops).

üî∏ Example 2: Repeat Loop for Blinking LED

  bit led = 0;

  initial begin
    $display("Blinking LED...");
    repeat (4) begin
      #5 led = ~led;
      $display("[%0t] LED = %0b", $time, led);
    end
    $display("LED blinking finished.");
  end
endmodule

üî∏ Example 3: Repeat Wait for Clock Edges

module tb;
  bit clk = 0;

  // Clock generation
  always #10 clk = ~clk; // 20ns period

  initial begin
    int delay_cycles = 3;
    $display("[%0t] Waiting for %0d rising edges of clk", $time, delay_cycles);
    repeat (delay_cycles) @(posedge clk);
    $display("[%0t] Done waiting for clk edges", $time);
    $finish;
  end
endmodule

üí¨ Simulation Timeline:
Time 0: Start
Time 10: 1st posedge
Time 30: 2nd posedge
Time 50: 3rd posedge
Time 50: Done waiting

üü® Comparison with for loop:

end

repeat version:

‚úÖ If you don‚Äôt need the value of i, repeat is shorter and cleaner.

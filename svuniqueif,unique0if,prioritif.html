How it works: If multiple conditions are true at the same time, it will only execute the first one.

‚ö° priority if

Purpose: The priority if ensures that only the first true condition gets executed. But, it works by considering the priority of each condition in order.

How it works: If you have multiple true conditions, it prioritizes the first one that matches.

üìò Example:

priority if (a > 10) begin
    $display("Condition A is true");
end else if (b > 10) begin
    $display("Condition B is true");
end else begin
    $display("None of the conditions matched");
end

üåç Real Life:

Think of a priority list for deciding what you want to eat:

  • If you feel hungry, you might choose a pizza üçï.
  • If pizza isn't available, you go for pasta üçù.
  • If both are available, you only pick the pizza (because it's higher on your priority list!).

üìù Summary:

  • unique if üèÖ: Only one condition can be true, and the first true condition will be executed.
  • unique0 if üö´: Ensures only one condition is true, but also handles the case where none are true.
  • priority if ‚ö°: Prioritizes conditions in order and executes the first one that is true.

These are used for handling multiple conditions in a way that avoids confusion and conflicts, making the code more predictable!

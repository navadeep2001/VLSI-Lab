How it works:

  • The loop body executes first (at least once).
  • Then the condition is checked.
  • If the condition is true, it runs again.
  • If the condition is false, it stops.

üå± Real-Life Example:

You go to a store and try to open the door. Whether it's locked or not, you try once before checking if the store is open.

    end while (store_open); // condition is false from start
  end
